# 환영합니다!

이 블로그는 **GitHub Pages**와 **마크다운**을 사용하여 만들어졌습니다. 별도의 서버 없이 GitHub 저장소만으로 블로그를 운영할 수 있습니다.

## 게시글 작성 방법

새 게시글을 작성하려면 `pages/` 폴더에 `.md` 파일을 추가하면 됩니다.

### Front Matter

각 게시글 상단에는 다음과 같은 **Front Matter**를 작성합니다:

```yaml
---
title: '게시글 제목'
date: 2025-11-28
tags: ['태그1', '태그2']
category: '카테고리'
description: '게시글 설명'
---
```

### 마크다운 문법

일반적인 마크다운 문법을 모두 사용할 수 있습니다.

#### 텍스트 스타일링

- **굵은 글씨**는 `**텍스트**`로 작성
- *기울임*은 `*텍스트*`로 작성
- `인라인 코드`는 백틱으로 감싸기

#### 코드 블록

코드 블록은 언어를 지정하면 자동으로 하이라이팅됩니다:

```javascript
function greet(name) {
  console.log(`안녕하세요, ${name}님!`);
}

greet('방문자');
```

```python
def greet(name):
    print(f"안녕하세요, {name}님!")

greet("방문자")
```

#### 인용문

> 마크다운은 읽기 쉽고 쓰기 쉬운 텍스트 포맷입니다.
> HTML로 변환하기도 매우 쉽습니다.

#### 목록

순서 없는 목록:
- 항목 1
- 항목 2
- 항목 3

순서 있는 목록:
1. 첫 번째
2. 두 번째
3. 세 번째

#### 링크

[GitHub](https://github.com)를 방문해보세요.

## 블로그 기능

이 블로그는 다음과 같은 기능을 제공합니다:

| 기능 | 설명 |
|------|------|
| 다크 모드 | 우측 상단 버튼으로 테마 전환 |
| 검색 | 게시글 제목과 내용 검색 |
| 태그 필터 | 태그별 게시글 필터링 |
| 댓글 | Giscus 기반 GitHub 댓글 |

## 배포 방법

1. 게시글을 작성합니다
2. `git push`로 변경사항을 푸시합니다
3. GitHub Actions가 자동으로 배포합니다
4. 잠시 후 블로그에 반영됩니다
